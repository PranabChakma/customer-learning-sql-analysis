# Customer Learning & Engagement Analysis (SQL)

## 📌 Project Overview
This project analyzes student learning behavior using the 365 Data Science customer behavior dataset.
The goal is to extract business-driven insights that support decision-making related to course quality,
student engagement, certification sales, and career track effectiveness.

## 🧠 Business Questions Answered
1. Which courses have the highest student engagement and best ratings?
2. Which courses have the highest student completion rates and what is the average time students spend on them?
3. Which students are most likely to purchase certificates?
4. How does exam performance relate to career track enrollment?
5. What distinguishes high-engagement students from low-engagement users?

## 🛠 Tools & Skills Used
- SQL (JOINs, GROUP BY, CASE, HAVING, Aggregations)
- Business Analytics
- Data-Driven Decision Making

## 📂 Project Structure
- `sql_queries/` → Interview-ready SQL queries
- `schema/` → Database schema reference
- `dataset/` → Dataset dictionary (for reference)

## 🚀 Key Insights
- High-engagement courses drive both learning time and ratings
- Students with consistent learning behavior are more likely to buy certificates
- Exam success strongly correlates with career-track enrollment

## 📎 Dataset
Dataset provided by 365 Data Science (used for educational purposes).
